# 🔁 Idea → TODO Promotion Checklist

Use this checklist before moving anything from `IDEAS.md` to `TODO.md`.

---

## ✅ Promotion Criteria

- [ ] The problem is clearly stated
- [ ] The value is understood
- [ ] Scope is roughly defined
- [ ] This aligns with current project goals
- [ ] This will not block higher-priority work

---

## 🧱 Architecture & Process

- [ ] ADR required?
- [ ] Affected components identified
- [ ] Backward compatibility considered
- [ ] Security / data implications reviewed

---

## 🧪 Delivery Readiness

- [ ] Acceptance criteria written
- [ ] Tests are feasible
- [ ] Docs impact understood
- [ ] Timebox or milestone identified

---

If all boxes are checked → **promote to TODO.md** 🎯
